Apa itu DOM?

DOM (Document Object Model) adalah sebuah interface pemrograman untuk dokumen HTML dan XML. DOM merepresentasikan dokumen tersebut dalam bentuk struktur pohon (tree structure) di mana setiap node pada pohon tersebut merupakan objek yang mewakili bagian-bagian dari dokumen, seperti elemen, atribut, dan teks.
